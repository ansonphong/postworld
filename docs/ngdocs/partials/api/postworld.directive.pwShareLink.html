pwShareLink
directive in module postworld

Description

Generates a share link for a particular post ID.

Usage

as attribute
<ANY pw-share-link="{expression}"
     share-link-post-id="{string}"
     share-link-dynamic="{boolean}">
   ...
</ANY>

Parameters

ParamTypeDetails
pwShareLinkexpression

The expression to map the share link to

shareLinkPostIdstring

Required. The ID of the post which to generate the share link for

shareLinkDynamicboolean

If true, sets up a watch on the value of shareLinkPostId

Example

<div pw-share-link="shareLink" share-link-post-id="post.ID">{{shareLink}}</div>
