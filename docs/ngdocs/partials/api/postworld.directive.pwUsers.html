pwUsers
directive in module postworld

Description

Provides methods to access user data.

Usage

as attribute
<ANY pw-users
     user-id="{string}"
     user-model="{expression}"
     user-dynamic="{string|boolean}">
   ...
</ANY>

Parameters

ParamTypeDetails
userIdstring

A user ID to retreive data for. If no user is provided, all available users are provided.

userModelexpression

Where to bind the user.

userDynamicstringboolean

If the user ID is expected to change.
