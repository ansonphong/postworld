pwInclude
directive in module postworld

Description

Used to include a Postworld template partial within an isolated scope. Post and meta data can be easily make available in the template.

Usage

as attribute
<ANY pw-include="{string}"
     include-vars="{expression}"
     include-meta="{expression}"
     include-post="{expression}"
     include-enable="{boolean}"
     include-class="{string}">
   ...
</ANY>

Parameters

ParamTypeDetails
pwIncludestring

Psuedo-path to the template. Use panels/widget to include templates/panels/widget.html

includeVarsexpression

Vars to assign within the include as $scope.vars

includeMetaexpression

Object to be assigned as $scope.meta within the included template

includePostexpression

Object to be assigned as $scope.post within the included template

includeEnableboolean

Dynamic. Whether or not to actually enable the load the include Can be used to prevent the template from loading in certain instances.

includeClassstring

Class(es) to be added to the include element

Example

<div pw-include="galleries/gallery-frame" include-post="post"></div>
