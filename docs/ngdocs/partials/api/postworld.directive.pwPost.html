pwPost
directive in module postworld

Description

Loads a post template and injects it with data.

Usage

as attribute
<ANY pw-post
     post-id="{string}"
     post-view="{string}"
     post-query="{object}"
     post-class="{string}"
     post-loading="{string}">
   ...
</ANY>

Parameters

ParamTypeDetails
postIdstring

Optional. Post ID

postViewstring

Which registered view template to use.

postQueryobject

Query vars to derive post from.

postClassstring

Optional. Classes to add to the template element.

postLoadingstring

Optional. An expression to assign a loading boolean.
