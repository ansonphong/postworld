pwEval
directive in module postworld

Description

Evaluates a string as javascript at the time of loading. Works well for initializing third-party libraries.

Usage

as attribute
<ANY pw-eval="{string}"
     eval-timeout="{number}"
     eval-context="{string}"
     eval-watch="{expression}">
   ...
</ANY>

Parameters

ParamTypeDetails
pwEvalstring

A string to evaluate as Javascript

evalTimeoutnumber

Optional. Milliseonds to timeout before evaluating

evalContextstring

Optional. Context in which to evaluate. Options : 'scope' / 'window'. Default : 'scope'

evalWatchexpression

Optional. An expression to watch for changes, on change re-evalute.
