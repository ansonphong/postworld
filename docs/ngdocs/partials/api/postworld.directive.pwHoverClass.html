pwHoverClass
directive in module postworld

Description

Adds specified class(es) to an element on mouseover, and removes the classes on mouseleave.

Usage

as attribute
<ANY pw-hover-class="{string}"
     hover-on-delay="{number}"
     hover-off-delay="{number}">
   ...
</ANY>

Parameters

ParamTypeDetails
pwHoverClassstring

Class(es) to add when hovered.

hoverOnDelaynumber

Optional. Milliseconds after mouseover before classes are added.

hoverOffDelaynumber

Optional. Milliseconds after mouseleave before classes are removed.
