Class: pwHoverClass

pwHoverClass

Adds specified class(es) to an element on mouseover, and removes the classes on mouseleave.

Constructor

new pwHoverClass(pwHoverClass, hoverOnDelay, hoverOffDelay)

Parameters:
Name Type Description
pwHoverClass string Class(es) to add when hovered.
hoverOnDelay number Optional. Milliseconds after mouseover before classes are added.
hoverOffDelay number Optional. Milliseconds after mouseleave before classes are removed.
