Class: liveFeed

liveFeed

Generates a sequences of posts.

Constructor

new liveFeed(feedId)

Generates a feed from predefined feed paramters within the specified element.
Parameters:
Name Type Description
feedId string A predefined feed ID

<script>
pw.feeds['newsFeed'] = {
	preload: 10,
	load_increment: 10,
	offset: 0,
	order_by: '-post_date',
	view: {
		current: 'list'
	},
	query:{
		post_type:['post'],
		post_status:'publish',
		posts_per_page: 100
	}
};
</script>
