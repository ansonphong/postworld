Class: pwEval

pwEval

new pwEval(pwEval, evalTimeout, evalContext, evalWatch)

Evaluates a string as javascript at the time of loading. Works well for initializing third-party libraries.
Parameters:
Name Type Description
pwEval string A string to evaluate as Javascript
evalTimeout number Optional. Milliseonds to timeout before evaluating
evalContext string Optional. Context in which to evaluate. Options : 'scope' / 'window'. Default : 'scope'
evalWatch expression Optional. An expression to watch for changes, on change re-evalute.
