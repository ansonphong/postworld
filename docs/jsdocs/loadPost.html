loadPost

new loadPost(postId, postView, postQuery, postClass, postLoading)

Loads a post template and injects it with data.

postId String Optional. Post ID
postView String Which registered view template to use.
postQuery Object Query vars to derive post from.
postClass string Optional. Classes to add to the template element.
